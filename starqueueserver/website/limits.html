StarQueue limits.



  • Send up to 50,000 messages per month.
  • Maximum queue size 1,000 unreceived messages.
  • Maximum size of the MessageBody field is 262144 bytes (256KB).
  • MessageRetentionPeriod (unreceived messages are deleted after this amount of time) is 4 hours.

Exceeding limits:
StarQueue returns status 429 "Too Many Requests" when queue message sends is exceeded.
